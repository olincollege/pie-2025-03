1. Run the Arduino code by plugging the Arduino Mega into the laptop
2. Disconnect the USB from the computer and plug it into a Raspberry Pi.
3. The Raspberry Pi should have Raspberry Pi Connect installed to control the screen with your computer (since the display is not touchscreen)
4. Open Thonny and use the display code for the Raspberry Pi.
5. Press F11 to put Thonny in full-screen mode for a better typing experience.
